## Project Overview
This project analyzes coffee shop sales data using Microsoft Excel to uncover trends, patterns, and business insights.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables & Pivot Charts
- Slicers & Timelines
- Excel Formulas (SUMIFS, IF, XLOOKUP)

## Key Insights
- Identified peak sales hours and busiest days
- Found top-selling products and categories
- Analyzed monthly sales trends
- Compared store-wise performance
